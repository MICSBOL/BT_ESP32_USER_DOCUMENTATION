\chapter{Componentes Requeridos}

\section{Requisitos de Hardware}

Para utilizar el sistema de Controlador RC necesitarás el siguiente hardware:

\begin{itemize}
    \item Placa de desarrollo ESP32
    \item Teléfono inteligente Android
    \item Módulo driver para motor (L298N, BTS7960 o similar)
    \item Motores DC o servos
    \item Fuente de alimentación o batería
    \item Cables jumper
\end{itemize}

\section{Componentes Opcionales}

El sistema también puede admitir dispositivos adicionales:

\begin{itemize}
    \item LEDs
    \item Zumbador (buzzer)
    \item Sensores I\textsuperscript{2}C
    \item Módulo de expansión GPIO MCP23017
    \item Sensores analógicos
\end{itemize}

\section{Requisitos de Software}

También necesitarás las siguientes herramientas de software:

\begin{itemize}
    \item IDE de Arduino
    \item Paquete de soporte de placa para ESP32
    \item Aplicación Android RC Controller
\end{itemize}
