\chapter{Customizing Your Project}

\section{Adding Sensors}

The firmware supports sensors connected to the ESP32.

Typical sensors include:

\begin{itemize}
	\item Distance sensors
	\item Temperature sensors
	\item Analog sensors
\end{itemize}

Sensor data can be sent back to the Android application using telemetry.

\section{Using I2C Devices}

Additional devices can be connected using the I2C interface.

Examples include:

\begin{itemize}
	\item MCP23017 GPIO expanders
	\item Environmental sensors
	\item Displays
\end{itemize}

\section{Expanding Outputs}

The system allows users to control multiple outputs.

Examples include:

\begin{itemize}
	\item LEDs
	\item Relays
	\item Motor drivers
	\item Buzzers
\end{itemize}
