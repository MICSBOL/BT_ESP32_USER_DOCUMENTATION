\chapter{Required Components}

\section{Hardware Requirements}

To use the RC Controller system you will need the following hardware:

\begin{itemize}
	\item ESP32 development board
	\item Android smartphone
	\item Motor driver module (L298N, BTS7960, or similar)
	\item DC motors or servos
	\item Power supply or battery
	\item Jumper wires
\end{itemize}

\section{Optional Components}

The system can also support additional devices:

\begin{itemize}
	\item LEDs
	\item Buzzer
	\item I2C sensors
	\item MCP23017 GPIO expansion module
	\item Analog sensors
\end{itemize}

\section{Software Requirements}

You will also need the following software tools:

\begin{itemize}
	\item Arduino IDE
	\item ESP32 board support package
	\item RC Controller Android application
\end{itemize}
