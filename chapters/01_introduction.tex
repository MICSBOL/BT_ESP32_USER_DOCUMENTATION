\chapter{Introduction}

\section{What is the ESP32 RC Controller System}

The ESP32 RC Controller system allows you to control electronics projects using an Android phone and an ESP32 microcontroller.

The Android application communicates with the ESP32 using Bluetooth. Control commands from the phone are sent to the ESP32, which then updates motors, servos, LEDs, and other devices connected to the board.

At the same time, the ESP32 can send information back to the phone, such as:

\begin{itemize}
	\item Sensor values
	\item Button states
	\item System status
	\item Real-time telemetry
\end{itemize}

This creates a bidirectional communication system between the Android application and the ESP32.

\section{Who This Guide is For}

This guide is intended for:

\begin{itemize}
	\item Students learning electronics
	\item Robotics hobbyists
	\item Makers building DIY projects
	\item Beginners using the ESP32 platform
\end{itemize}

Only basic knowledge of the Arduino IDE and simple electronic wiring is required.

\section{What You Can Build}

Using the RC Controller system you can build many types of projects:

\begin{itemize}
	\item RC cars
	\item Robots
	\item Camera pan systems
	\item Sensor monitoring platforms
	\item Bluetooth-controlled devices
\end{itemize}

The firmware was designed to be modular, allowing users to extend the system with additional sensors, motors, and control devices.
